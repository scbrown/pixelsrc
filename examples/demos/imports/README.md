# Import Demo Fixtures

PNG test images for demonstrating the `import_png()` functionality.

## Fixtures

These fixtures are auto-generated by the demo tests on first run:

| File | Description |
|------|-------------|
| `basic_4x4.png` | 4x4 pixel art with 4 distinct colors |
| `gradient_8x8.png` | 8x8 gradient for palette detection testing |
| `spritesheet_16x4.png` | 16x4 horizontal spritesheet (4 frames) |
| `transparent_4x4.png` | 4x4 checkerboard with transparency |

## Demo Tests

Located in `tests/demos/imports/mod.rs`:

- **png_to_jsonl** - Basic PNG to JSONL conversion
- **palette_detection** - Auto-detect palette from image colors
- **multi_frame_import** - Import spritesheet as animation frames
- **transparency_handling** - Preserve transparent pixels with `{_}` token

## Verification

```bash
cargo test demos::imports
```
